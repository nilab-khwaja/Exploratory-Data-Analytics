# ğŸ“– EDA on Unicorn Companies <a name="about-project"></a>

This project is an **Exploratory Data Analysis (EDA)** focused on unicorn companiesâ€”startups valued at $1 billion or more. The goal of this project is to derive actionable insights using the **DIKW Framework** (Data, Information, Knowledge, Wisdom). The analysis includes valuation trends, funding patterns, industry distributions, geographical breakdowns, and investor impact.

<!-- Features -->

### Key Features <a name="key-features"></a>

Key features of this analysis include:

- Insightful visualizations of unicorn companies by valuation, funding, and geography.
- Identification of top-performing industries and their growth patterns.
- Recommendations for startups, investors, and policymakers using the DIKW framework.
- Insights into key investor contributions and funding strategies.
